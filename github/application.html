而在這些過程中，無論是了解數據來源、處理數據的過程、搜索公式並試行直到取得最終資料，都比以往處理其他業務來說更投入及更有想解決問題的動力以及成就感。所以開始尋找與這類相關的工作後，也漸漸的開始對於數據分析或是軟體工程有初步的思考，想著自學Python試試看是否能適應或者會不會很排斥，就看見了彭彭老師的學習營，並且了解學習營後感受到規劃很扎實，學習的最後也能自行選擇往前端或是後端發展並不受限，這個了解的過程中逐漸對學習營有信任感及興趣。
而也因為軟體工程師的起薪以及發展，相較於其他職業來說也是比較好的，也考量到本身性格相較於內向，且對於長時間待在電腦前的作業模式也很喜歡和習慣，所以以初步可以考慮到的層面對我來說是順勢而為，並不會有像考量其他轉職方向時的排斥感(例如業務等有薪資成長幅度的職業)。並且在學習架設網站過程也覺得滿有趣及挑戰，所以決定申請此學院。

·如何看待人工智慧的發展對軟體工程師的影響？
人工智慧提供了許多資源，甚至可以寫程式，而人工智慧發展也不會停止，會有更多不同的應用程式和新技術，或許也會取代一部分的軟體工程師。
首先需要扎穩基本功，非常熟程式語言的邏輯，此後再來妥善利用人工智慧的功能，例如可利用人工智慧加快學習新語言的速度，或者在寫程式過程解不出來的bug，可以一邊查找網路資料及利用人工智慧檢核，像是一個小團隊般的進行合作，但除此之外也要能分辨出AI提供的資訊是否正確，再加以利用。若熟悉且運用得當的話，對軟體工程師是一份加分項目，可以更有效率地產出，以及團隊合作及溝通能力也會是必備的技能。
